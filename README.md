# Discord_Bot
디스코드 봇을 제작합니다.


## 가능 명령어

+ 엄: 준식 출력

+ !더하기 (숫자1) (숫자2) ... (숫자n): 해당 숫자들을 모두 더함

  입력: !더하기 1 3 4 5.1
  
  *출력: 13.1*


+ !빼기 (숫자1) (숫자2) ... (숫자n): 해당 숫자들을 모두 뺌

  입력: !빼기 50 20 30 1
  
  *출력: -1*


+ !계산 (+-*/로 이루어진 다항식): 헤딩 다항식을 계산

  입력(평범한 다항식): !계산 10+40*2

  입력(+-형태의 다항식): !계산 50+30+20+-10

  입력(곱셈만): !계산 30*3

  입력(나누기만): !계산 180/2
  
  *출력: 90* 

  입력(나머지가 있는 나누기): !계산 3/2

  *출력: 1.5*
