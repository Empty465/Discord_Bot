# Discord_Bot
여러 명령어들이 있는 디스코드 봇 입니다.

## 실행 방법

<details>
<summary>이곳을 눌러 펼칠 수 있습니다.</summary>

0. 이 페이지에서 <> 코드를 눌러 git을 통해 파일들을 다운받거나 .ZIP 다운로드하기를 선택한 뒤에 압축을 풉니다.
1. <a href="https://www.python.org/">이곳을 눌러</a> 하드웨어 환경에 맞는 파이썬을 설치합니다.
2. <a href="https://docs.google.com/document/d/1r_iXdeoNrKRi09DgBxmQW6x-0zcv0-2jcga-Tc7t8mI/edit?usp=sharing">이곳을 눌러</a> 가이드에 따라 디스코드 봇을 생성하고, 토큰을 main.py가 위치한 같은 폴더 안에 token.txt를 만들고 토큰을 붙혀넣어 저장합니다. (1번에서 7번까지의 내용)
3. 해당 디스코드 봇을 사용할 디스코드 서버에 만든 봇을 초대합니다. (가이드 8번부터의 내용)
4. (윈도우 기준) Win+R 을 눌러 검색창을 띄운뒤에 CMD를 검색하고 실행합니다.
5. CMD에 `pip install -U discord.py` 명령어를 입력하여 디스코드 모듈을 설치받습니다. 만약 해당 명령어가 오류가 난다면 `py -m pip install -U discord.py` 를 대신 입력합니다.
6. main.py 파이썬으로 실행하여 결과 두 줄이 표시되는 것을 확인 합니다. (첫 실행에 안 될 수 있으니 안 된다면 다시 실행 바랍니다)
7. 디스코드 봇이 온라인 상태가 되었는지 확인하고, 온라인 상태가 되었다면 성공입니다.
8. 디스코드 봇이 온라인이 되었다면 아래 가능 명령어를 입력하면 작동됩니다. (!도움 을 입력하여 가능한 명령어를 확인 가능 합니다)

</details>

## 가능 명령어

!도움 을 입력하여 가능한 명령어를 확인할 수 있습니다

아래 항목을 클릭하여 펼칠 수 있습니다.

<details>
<summary>체크리스트</summary>

  + !체리추가 (내용): 해당 내용의 체크리스트 추가
  + !체리목록: 현재 체크리스트 조회
  + !체리완료 (번호): 해당 번호의 체크리스트 완료처리
  + !체리제거 (번호): 해당 번호의 체크리스트 제거

</details>

<details>
<summary>출석 기능</summary>

  + !출석: 해당 날짜에 출석 (+1 출석 점수. 다음날 00시 이후에 다시 가능)
  + !출석조회 (이름): 자신의 출석 횟수 조회, 이름이 들어가면 해당 대상 출석 횟수 조회
  + !출석순위: 자신을 포함한 출석순위 TOP 5 조회

</details>

<details>
<summary>시수 기록 기능</summary>

  + !시수 (이름): 해당 이름의 시수 출력 (이름대신 시간 입력시, 그 시수에 해당하는 이름들 출력)
  + !시수추가 (이름) (시간): 해당 이름에 입력된 시간만큼 시수 추가 (없는 이름이면 새로 추가)
  + !시수차감 (이름) (시간): 해당 이름에 입력된 시간만큼 시수 차감
  + !시수수정 (이름) (시간): 해당 이름의 시수를 입력한 시간으로 수정

</details>

<details>
<summary>사칙연산</summary>

+ !더하기 (숫자1) (숫자2) ... (숫자n): 해당 숫자들을 모두 더함

  입력: !더하기 1 3 4 5.1
  
  + *출력: 13.1*


+ !빼기 (숫자1) (숫자2) ... (숫자n): 해당 숫자들을 모두 뺌

  입력: !빼기 50 20 30 1
  
  + *출력: -1*


+ !계산 (사칙연산): 해당 식을 계산

  입력(평범한 사칙연산): !계산 10+40*2

  입력(+-형태의 사칙연산): !계산 50+30+20+-10
  
  + *출력: 90* 

  입력(나머지가 있는 나누기): !계산 3/2

  + *출력: 1.5*

  입력(괄호 섞인 식): !계산 3(2+4)-5(1.5+0.5)

  + *출력: 8*

</details>

<details>
<summary>기타 쓸모 없는 것들</summary>

  입력: 엄
  + *출력: 준식*

  입력: ㅋㅋ (또는 ㅋ로 끝나는 메시지)
  + *출력: ㅋㅋㅋ*

</details>
